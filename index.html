The Mission Pinball Framework (“MPF”) is a free Python-based pinball software framework that’s used to run real pinball machines. However, to build pinball machines you also need hardware which is where Mission Pinball Hardware ("MPH") comes in.

MPF contains PCBs to connect common hardware, 3D models for mechs and more.


Printed Circuit Boards for Your Machine

There is a lot of electronics needed to run a pinball machine. Some of it can be bought off-the-self but other parts are custom to the machine. We maintain a list of custom PCBs with schematics to customize and fabricate according to your needs. This includes light boards, optos, custom mechs, complete control boards and more.

Did you build some board worth sharing? Please let us know in the MPF User Forum and/or create a Pull Request for this repository.

2D and 3D CAD Models

As a pinball machine contains a lot of different parts below and above the playfield it makes sense to model everything in a CAD program. We maintain a list of CAD models which can be used to model standard parts. If possible we try to validate that those models are correct and note who used them already.

Additionally, we list a few models which are intended to be 3D printed. This contains holders, mounts and even complete mechs. Some are only intended for prototyping others could also be used in a final machine.

Did you create some model worth sharing? Please let us know in the MPF User Forum and/or create a Pull Request for this repository.

Physical Machine Building

Furthermore, the MPF documentation contains a section about wiring, physical layout and more about building a machine.

If you are interested more about the software side of things head over to the MPF Website. We also cover how to use various hardware platforms and existing machines and how to use all kinds of pinball mechs with MPF.
